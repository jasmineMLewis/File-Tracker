# File-Tracker
File Tracker, a VB.NET web application, allows staff to maintain accurate record and traction of destroyed or stored boxes and files, and permit users to requests files.

# Technology Used
<b>FrontEnd</b> <br />
HTML5, CSS3, Javasript <br />
<b>BackEnd</b> <br />
ADO.NET, SQL, VB6 <br />
